# Siri-Dhanya Hub 🌾

## Overview

Siri-Dhanya Hub is an Android application developed using Generative AI concepts to support millet farmers and consumers. The app provides live mandi prices, millet recipes, health benefits, and farmer connectivity to promote sustainable agriculture and healthy lifestyles.

## Features

* 📈 **Mandi Watch** – View millet market prices from different cities
* 🍲 **Recipe Lab** – Step-by-step millet recipes in Kannada
* ❤️ **Health Benefits** – Nutritional information about millets
* 🔍 **Search Functionality** – Search recipes by millet type
* ⭐ **Favorite Recipes** – Save favorite dishes
* 🤝 **Direct Buy** – Connect with local Farmer Producer Organizations (FPOs)

## Technologies Used

* Android Studio
* Kotlin / Java
* Firebase
* RecyclerView
* Glide Library
* SQLite / Room Database
* REST APIs
* Generative AI

## Project Goal

The main goal of the project is to:

* Promote millet consumption
* Help farmers track better market prices
* Encourage climate-friendly agriculture
* Improve nutrition awareness
